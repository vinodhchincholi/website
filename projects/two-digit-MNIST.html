Neural Network to recognize pairs of handwritten digits

  • Used images from MNIST to concatenate and form double digit images. Classification with 100 labels.
  • Fully connected neural network with 2 hidden layers, stochastic gradient descent optimizer on ‘cuda’ device.
  • We got an accuracy of 97% on validation set and 96% on test set. Possible improvements include using better optimizers and implement convolutional neural networks for the dataset.
